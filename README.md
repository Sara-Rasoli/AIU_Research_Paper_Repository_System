AIU Reaserch Paper Repository System

## 📌 Project Description
A web-based Research Management System developed using PHP and MySQL.
The system allows students to:
- Submit research projects
- Search and download submissions
- Generate reports and citations
- Manage student access

## 🛠 Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP

## 📂 Project Structure
- module1_student_access.php
- module2_submission.php
- module3_search_download.php
- module4_reports_citations.php
- db_connect.php

## 🗄 Database
Database file is included as:
`database_schema.sql`

Import it into phpMyAdmin before running the project.

## ▶ How to Run
1. Install XAMPP
2. Move project folder to htdocs
3. Import SQL file into phpMyAdmin
4. Open:
   http://localhost/project-folder-name
